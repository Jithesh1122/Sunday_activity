# Sunday Activity - Simple React Components

A React + Vite project built for the Sunday Activity task.

## Objective
Understand the basics of:
- React functional components
- JSX
- Component props
- Simple CSS styling

## Features Implemented
- `Header`, `Content`, and `Footer` functional components
- JSX-based UI structure
- Props passed to components
- Responsive styling with plain CSS
- "About Me" button linking to `https://jitheshshetty.vercel.app/`

## Tech Stack
- React
- TypeScript
- Vite
- CSS

## Project Structure
- `src/App.tsx` - root page composition
- `src/components/Header.tsx`
- `src/components/Content.tsx`
- `src/components/Footer.tsx`
- `src/style.css` - component/page styles

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
